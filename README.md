# Todo-List
- This project,To-Do List application is a simple and efficient management tool that allow user to create,edit,update,delete the tasks. 
## Features
- create
- edit
- update
- delete
- store history
  
 ## Languages
- python
- html,css
- django
- sqlite
  
## Requirements
- 4GB ram
- 256 ROM
- Python-3.08
- django==5.1.5
